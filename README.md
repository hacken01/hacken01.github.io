## ğŸ¨ Customization

### Colors

The theme uses Iron Man inspired colors defined in CSS variables:
- **Iron Red**: `#C41E3A` - Primary red color
- **Iron Gold**: `#4FC3F7` - Light blue accent color
- **Dark Background**: `#0A0A0A` - Primary background

Edit `assets/css/style.css` to customize colors:

```css
:root {
    --iron-red: #C41E3A;
    --iron-gold: #4FC3F7;
    --bg-primary: #0A0A0A;
    /* ... other variables */
}
```

## ğŸš€ Deployment

### GitHub Pages

This site is automatically deployed to GitHub Pages when you push to the `main` branch.

1. Push your changes:
   ```bash
   git add .
   git commit -m "Update portfolio"
   git push origin main
   ```

2. Enable GitHub Pages:
   - Go to your repository settings
   - Click on "Pages" in the sidebar
   - Under "Source", select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Your site will be live at `https://hacken01.github.io`

### Other Hosting Options

- **Netlify**: Drag and drop the folder or connect your Git repository
- **Vercel**: Import your Git repository
- **Cloudflare Pages**: Connect your Git repository
- Any static hosting service
